# Agent Rook

**Brief your agent. It handles the rest.**

An open-source AI agent framework for solo operators. Drop in your knowledge, configure your dashboard, deploy your own AI-powered business assistant.

- Model agnostic (Claude, GPT, Gemini)
- Pluggable knowledge bases (just markdown files)
- Pre-built dashboard (hub-and-spoke, configurable)
- Built-in tools (scheduling, tracking, reports, compliance)
- One-command setup: `npx create-agentrook`

Built by [@kellsmith](https://github.com/kellsmith) — a daycare owner who built an AI assistant for her business, then realized the engine works for everyone.
